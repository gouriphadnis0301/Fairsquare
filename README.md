# Voting System Using Blockchain

A secure and transparent voting system built using **Django** and **blockchain technology**. This project simulates a real-world election process with tamper-proof vote records.

## Features

- **User Authentication** via Aadhar number.
- **Secure Voting** using blockchain principles.
- **Vote Tracking** to prevent multiple votes by the same user.
- **Admin Panel** for managing voters and monitoring votes.
- **Candidate Details** displayed before casting vote.
- **Responsive UI** with clear voting workflow.

## Technology Stack

- **Backend:** Django (Python)
- **Database:** SQLite (default), can be switched to PostgreSQL/MySQL
- **Frontend:** HTML, CSS, Bootstrap
- **Blockchain Logic:** Custom Python implementation for vote integrity
